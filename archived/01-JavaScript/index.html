Objects
Accessing properties of an object: 
  • obj.name = "Simon";
var name = obj.name;
                        
  • obj["name"] = "Simon";
var name = obj["name"];
                        
  • obj.for = "Simon"; // Syntax error,
//because 'for' is a reserved word
obj["for"] = "Simon"; // works fine
Objects
Can be initialized as: 
  • var obj = {
    name: "Carrot",
    "for": "Max",
    details: {
    color: "orange",
    size: 12
    }
}
Attributes can be chained like: 
  • > obj.details.color
orange
> obj["details"]["size"]
12
Arrays
  • Arrays are a special type of object
  • Magic property called 'length'
  • 'length' is always one more than the last index
  • Demo:
Arrays
Iterating over Arrays 
  • for (var i = 0; i < a.length; i++) {
    // Do something with a[i]
}
  • for (var i = 0, len = a.length; i < len; i++) {
    // Do something with a[i]
}
  • for (var i = 0, item; item = a[i++];) {
    // Do something with item
}
  • for (var i in a) {
    // Do something with a[i]
}
